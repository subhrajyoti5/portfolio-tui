# @subhrajyoti5/portfolio

Subhrajyoti's TUI Portfolio — A full-screen terminal application.

## Quick Start

The simplest way to run this is using `npx` (requires Node.js):

```bash
npx @subhrajyoti5/portfolio
```

## Installation

You can also install it globally to run it anytime with the `subhra` command:

```bash
npm install -g @subhrajyoti5/portfolio
```

Then just run:
```bash
subhra
```

## Features
- Interactive Terminal UI
- No dependencies required (other than Node.js)
- Runs natively on Windows, Linux, and macOS

---
Built with C++ and [FTXUI](https://github.com/ArthurSonzogni/FTXUI).
